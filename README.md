 🎧 Simple Audio Player

A simple and responsive web-based audio player built using **HTML, CSS, and JavaScript**.  
It allows users to play, pause, skip songs, and search through a playlist.

---

 🚀 Features

- ▶️ Play / Pause audio
- ⏭️ Next / Previous song controls
- 🔍 Search songs from playlist
- 💾 Remembers last played song (using localStorage)
- 🎨 Simple and clean UI


 🛠️ Built With

- HTML5
- CSS3
- JavaScript
