🎵 Dynamic Music Player with Live API Search

An advanced, responsive web-based audio player built using **HTML, CSS, and JavaScript**. This player features pre-loaded online tracks for instant playback and integrates a live music API to search and stream any song dynamically from the internet.

---

## ✨ Features

* **📦 Pre-loaded Tracks:** Comes with 4 popular default tracks loaded directly via online streaming links, available instantly upon page load.
* **🔍 Live Online Search:** Integrated with a high-speed music API (Saavn API) allowing users to search for any Desi or International song.
* **⚡ Dynamic Playlist Generation:** Search results automatically update the UI playlist suggestions in real-time.
* **📱 Fully Responsive Design:** Clean and interactive user interface optimized for both desktop and mobile viewports.
* **🎛️ Audio Controls:** Features full play/pause toggles, next/previous track skipping, live progress bar seeking, and smooth volume control.

---

## 🛠️ Tech Stack Used

* **Frontend:** HTML5 (Semantic Structure), CSS3 (Custom Styling & Glassmorphism UI)
* **Logic & API Handling:** Modern JavaScript (ES6+ Async/Fetch API)
* **External API:** Saavn Music Developer API (For real-time streamable audio data fetching)
